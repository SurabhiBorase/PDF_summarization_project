# What this Does ? :--
# 1. Extracts text from a PDF.
# 2. Summarizes the extracted text using extractive summarization.
# 3. Prints both the extracted text and the summary.

Flask- 	Web application framework (file uploads, routing, rendering pages)
PyPDF2-	Extracting text from PDFs
nltk-	Tokenizing sentences, removing stopwords (extractive summarization)
collections-	Word frequency calculation (ranking sentences)
re (Regex)-	Text cleaning (removing special characters, fixing broken words)
transformers-	Using BART model for AI-generated abstractive summaries
torc=h	Deep learning computations for transformers (BART model)
os-	Managing uploaded PDF files
Jinja2-	Rendering dynamic HTML pages

Run Code -

source venv/bin/activate
python app.py
